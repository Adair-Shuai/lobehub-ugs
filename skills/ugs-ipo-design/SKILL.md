# 储气库注采方案辅助设计

## 概述

本 skill 提供地下储气库注采方案的标准化设计流程，
涵盖单井配产配注、多周期注采策略、调峰能力评估和井网优化。

## 适用场景

- 新建储气库注采方案设计
- 已有储气库扩容后方案调整
- 季节调峰方案制定
- 应急调峰能力评估
- 井网加密与调整

## 设计流程

### 第一步：需求分析

明确储气库运行目标：

```
- 工作气量需求 (WGV_target, 10^8 m³)
- 最大日采气量要求 (Qmax_w, 10^4 m³/d)
- 最大日注气量要求 (Qmax_i, 10^4 m³/d)
- 注气期长度 (Ti, 天, 通常 4月-10月 ≈ 210天)
- 采气期长度 (Tw, 天, 通常 11月-3月 ≈ 150天)
- 应急调峰启动时间 (t_start, 小时)
- 最低井口外输压力要求 (Pwh_min, MPa)
```

### 第二步：单井产能评价（节点分析）

**采气能力 — Inflow Performance**：

```
指数式产能方程：
q = C × (Pr² - Pwf²)^n

其中：
- C: 产能系数 (10^4 m³/d/MPa^(2n))
- n: 湍流指数 (0.5 < n ≤ 1.0)
- Pr: 地层压力 (MPa)
- Pwf: 井底流压 (MPa)
```

**一点法试井快速评估**：
```
q_AOF = q × [6 / (√(1 + 48 × (1 - α)) - 1)]
其中 α = (Pr² - Pwf²) / Pr²
```

**注气能力 — 注气压差分析**：
```
q_inj = C_inj × |Pwf - Pr|^n
```
注意：注气时需考虑地层破裂压力上限，Pwf_max = 0.9 × Pfracture

### 第三步：井数确定

```
采气工况井数：
N_w = Qmax_w / (q_w_single × R_run)

注气工况井数：
N_i = Qmax_i / (q_i_single × R_run)

总井数取二者最大值：
N = max(N_w, N_i)

其中 R_run ≈ 0.85-0.90 (运行效率，考虑检修、故障)
```

### 第四步：注采周期策略

**日注/采气量分配**：

```
注气期（210天典型分配）：
  4月:  5% 月工作气量  (低压启动)
  5月:  12%
  6月:  15%
  7月:  15%
  8月:  15%
  9月:  13%
  10月: 15%  (末期减速避免超压)
  另补 10% 作为缓冲余量

采气期（150天典型分配）：
  11月: 10% 月工作气量
  12月: 25%  (供暖高峰)
  1月:  30%  (供暖高峰)
  2月:  20%
  3月:  10%
  剩余 5% 为应急储备
```

### 第五步：井网布置原则

**井距计算**：

```
对于圆形均质储层：
d_min ≥ 2 × √(A / (π × N))

考虑井间干扰：
d_opt = max(2 × re, d_min)

其中 re 为单井泄气半径：
re = 0.029 × √(k × t / (φ × μ × ct))
```

**井网类型选择**：

| 储层条件 | 推荐井网 | 理由 |
|----------|----------|------|
| 构造简单、均质 | 面积井网 | 均衡采出 |
| 条带状储层 | 行列井网 | 顺物源方向 |
| 裂缝发育 | 沿裂缝方向部署 | 避免气窜 |
| 层状储层 | 分层系开发 | 提高纵向动用 |
| 含边底水 | 顶部布井 | 延缓水锥 |

### 第六步：多周期运行策略

```
周期 1 (启动)：垫底气注入 → 逐步增压
周期 2-3 (扩容)：逐步提高工作气量至设计值
周期 4+ (稳定运行)：设计工作气量 ± 5% 波动
周期 N (后期)：评估库容损失，必要时调整
```

关键监控指标：
- 气井产能递减率（< 5%/周期为正常）
- 水气比变化趋势
- 地层压力恢复速度
- 井间压力干扰程度
