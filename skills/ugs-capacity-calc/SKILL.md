# 储气库库容量与工作气量计算

## 概述

本 skill 提供地下储气库（UGS）库容量、工作气量、垫底气量的标准化计算流程，
适用于枯竭油气藏型储气库的库容参数评估。

## 适用场景

- 储气库建设可行性研究中的库容参数估算
- 已有储气库扩容潜力评估
- 多周期运行后库容变化分析
- 垫底气量优化与惰性气替代方案评估

## 计算流程

### 第一步：基础参数收集

收集以下关键参数：

```
- 储层原始压力 Pi (MPa)
- 储层原始温度 Ti (°C)
- 天然气偏差系数 Zi (无量纲)
- 储层孔隙体积 Vp (10^4 m³)
- 原始含气/含水饱和度
- 废弃压力 Pa (MPa)
- 最大允许运行压力 Pmax (MPa)
- 最小允许运行压力 Pmin (MPa)
```

### 第二步：原始地质储量计算（容积法）

```
G = 0.01 × A × h × φ × Sg / Bgi
```

其中：
- G — 天然气原始地质储量 (10^8 m³)
- A — 含气面积 (km²)
- h — 有效厚度 (m)
- φ — 有效孔隙度（小数）
- Sg — 含气饱和度（小数）
- Bgi — 原始天然气体积系数

### 第三步：P/Z 曲线法估算库容

```
P/Z = (Pi/Zi) × (1 - Gp/G)
```

绘制 P/Z vs Gp 关系曲线：
- 直线段斜率反映动态储量
- 外推至 P/Z = 0 得到最大可动储量
- 异常高压储层需校正（Roach 法或 Ramagost-Farshad 法）

### 第四步：最大库容量计算

```
C_max = G × (1 - Pa/Za × Zi/Pi)
```

废弃压力 Pa 取决于：
- 气井最低井口外输压力
- 井筒压力损失
- 压缩机吸入压力要求

### 第五步：工作气量计算

```
WGV = G × (Pmax/Zmax - Pmin/Zmin) / (Pi/Zi)
```

### 第六步：垫底气量计算

```
Cushion = C_max - WGV
```

## 检查清单

- [ ] P/Z 曲线线性段是否足够（R² > 0.95）
- [ ] 异常高压是否已校正
- [ ] 水侵影响是否已考虑（Cole 图版）
- [ ] 多周期的储层伤害是否已评估
- [ ] 垫底气经济性是否已分析（惰性气替代比例）
- [ ] 地面集输系统能力是否匹配最大日采气量
