# 储气库井筒完整性分析

## 概述

本 skill 提供地下储气库井筒完整性评价的系统化方法，
基于 NORSOK D-010 井屏障理论，涵盖设计、施工、运行和废弃全生命周期。

## 适用场景

- 储气库注采井完整性评价
- 老井再利用可行性评估
- 环空带压诊断与处理方案
- 套管腐蚀监测与剩余强度评价
- 弃置井封堵方案设计

## 分析流程

### 第一步：井屏障元素（WBE）识别

按 NORSOK D-010 识别两级屏障：

**一级屏障（Primary Barrier）**：
- 生产套管 + 固井水泥环（隔离地层流体）
- 生产封隔器
- 油管柱 + 井下安全阀（SCSSV）

**二级屏障（Secondary Barrier）**：
- 技术套管 + 固井水泥环
- 套管头密封总成
- 采气树主阀

### 第二步：井屏障状态评估

对每个 WBE 检查以下项目：

```
─ 是否存在？
─ 是否经过验证（试压/测井）？
─ 失效模式是否已识别？
─ 降级后的剩余能力？
```

**压井状态分类**：

| 状态 | 条件 | 措施 |
|------|------|------|
| 完好 | 两级屏障均有效 | 正常注采 |
| 降级 | 一级屏障失效，二级有效 | 限期修复 |
| 失效 | 两级屏障均失效 | 立即停产 |

### 第三步：环空带压分析

```
环空压力分类（API RP 90）：
- A 环空：油管-生产套管之间
- B 环空：生产套管-技术套管之间
- C 环空：技术套管-表层套管之间
```

**诊断步骤**：
1. 持续环空压力（SCP）vs 热膨胀压力 → 泄压/恢复测试
2. 气体组分分析 → 判断气源层位
3. 声波/噪声测井 → 定位泄漏点
4. 环空压力恢复曲线 → 估算泄漏速率

**可接受环空压力**：
```
P_acceptable = min( 套管抗内压强度的 50%,  地层破裂压力的 80%,  井口额定工作压力 )
```

### 第四步：套管/油管腐蚀评价

**腐蚀速率计算（De Waard-Milliams 模型）**：

```
log(V_cor) = 5.8 - 1710/T + 0.67×log(P_CO2)
```

其中 V_cor 单位为 mm/year, T 为开尔文温度, P_CO2 为 CO2 分压 (bar)

**剩余强度评估**（ASME B31G 或 DNV-RP-F101）：
- 均匀腐蚀 → 壁厚减薄直接计算
- 点蚀 → ASME B31G 有效面积法
- 裂纹 → API 579 FFS Level 1/2/3

### 第五步：固井质量评价

- CBL/VDL：评价一二界面胶结质量
- 超声成像（USIT/IBC）：360°成像评估窜槽
- 扇区水泥胶结测井（SBT）：分区定量评价

| CBL 振幅 | 胶结评价 | 措施建议 |
|----------|----------|----------|
| < 10% | 优 | 无需处理 |
| 10-30% | 中等 | 加强监测 |
| > 30% | 差 | 挤水泥修补 |

### 第六步：注采交变应力分析

储气库井的独有风险 — 周期性压力/温度变化导致：

1. **套管疲劳**：循环应力幅值 vs S-N 曲线
2. **水泥环微环隙**：套管膨胀-收缩导致脱粘
3. **管柱屈曲**：温度变化引起轴向力变化
   ```
   ΔF = E × A × α × ΔT
   ```

## 风险评估矩阵

| 概率\\后果 | 轻微 | 中等 | 严重 | 灾难 |
|-----------|------|------|------|------|
| 几乎确定 | M | H | VH | VH |
| 很可能 | M | M | H | VH |
| 可能 | L | M | H | H |
| 不太可能 | L | L | M | H |
| 罕见 | L | L | L | M |

H = 高风险（立即行动）, M = 中风险（计划行动）, L = 低风险（持续监测）
