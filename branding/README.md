# 品牌资源目录

将你的品牌资源文件放在此目录下：

| 文件 | 尺寸建议 | 说明 |
|------|---------|------|
| `logo.svg` | 120×40 | 主 LOGO，显示在侧边栏顶部 |
| `favicon.ico` | 32×32 | 浏览器标签页图标 |
| `og-image.png` | 1200×630 | 社交媒体分享预览图 |

> SVG 格式优先（支持响应式缩放），PNG 备选。

## 快速生成占位 LOGO

如果暂时没有设计稿，可以用在线工具快速生成：
- https://www.svgrepo.com/
- https://logo.com/
- 或直接在 LobeChat 里让 AI 生成一个 SVG LOGO
